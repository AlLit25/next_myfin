# LiVi Analytics

Додаток створений для автоматичного розрахунку загальних витрат.

## Логіка роботи балансу

У додатку реалізовано два типи балансу: **фактичний** та **математичний**.

- **Фактичний баланс** — це баланс, який вказується безпосередньо під час заповнення сум, де зберігаються кошти у гривні. Фактичний баланс є сумою всіх цих значень.
- **Математичний баланс** — це баланс, який починає розраховуватися, коли користувач вносить у додаток витрати або доходи. З цього моменту баланс стає математичним, оскільки немає впевненості, що користувач правильно вніс усі дані.

Щоб повернути математичний баланс до статусу фактичного, необхідно повторно заповнити суми, де зберігаються кошти.  
Цей циклічний процес рекомендовано повторювати **один раз на тиждень**.

## Можливості додатка

- Додавання доходів та витрат за категоріями
- Налаштування та перевірка фактичного балансу
- Додаткова можливість вказати суму заощаджень в USD (конвертується в гривні за поточним курсом)
- Зрозуміле представлення інформації у вигляді таблиці
- Редагування та видалення помилкових даних з таблиці статистичної інформації
- Мобільна (скорочена) версія
- Можливість створити PWA  
  *(Примітка: повідомлення та нагадування додаток не надсилає)*
